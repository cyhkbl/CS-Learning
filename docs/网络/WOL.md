# WOL

> 远程开机（Wake On Lan）通常只支持 **有线连接**。如果电脑是用 Wi-Fi 上网的，大部分主板不支持无线唤醒（WowLAN）。

### 主板 BIOS 设置

1. **进入 BIOS**：重启电脑，连续按下 `Delete` 或 `F2`（根据主板型号可能不同）。
2. **开启唤醒项**：进入“高级 (Advanced)”或“电源管理 (Power Management)”菜单。
3. **寻找关键字**：找到类似 `Wake on LAN`、`Power On By PCI-E`、`Remote Wake Up` 或 `PCIE Devices Power On` 的选项，将其设为 **Enabled**。
4. **关闭节能模式**：如果看到 `ERP Ready` 或 `Deep Sleep Mode`，将其设为 **Disabled**，否则关机后网卡会断电。

### Windows 系统设置

1. **修改网卡属性**：
* 右键“此电脑” -> “管理” -> “设备管理器”。
* 找到“网络适配器”，右键你的 **有线网卡**（通常是 Realtek 或 Intel Ethernet），选择“属性”。
* 在**电源管理**页：勾选“允许此设备唤醒计算机”和“只允许魔术封包 (Magic Packet) 唤醒计算机”。
* 在**高级**页：找到“魔术封包唤醒 (Wake on Magic Packet)”或“关机网络唤醒”，确保它是 **开启/启用** 状态。


2. **关闭快速启动**：
* 控制面板 -> 电源选项 -> 选择电源按钮的功能。
* 点击“更改当前不可用的设置”，**取消勾选**“启用快速启动”。**如果不关闭这个，Win11/10 关机后会进入深度休眠，导致网卡不响应。**

### 发送唤醒信号

1. 获取电脑身份信息：记录下电脑的 **MAC 地址**。
* 在电脑上打开 CMD，输入 `ipconfig /all`。
* 找到有线网卡的“物理地址”（例如：`AA-BB-CC-DD-EE-FF`）。

2. 在手机软件上唤醒。
