# 无线电

## 波段

### 中波（AM，MW）

中波电台的频率范围一般在522kHz ~ 1620kHz。由于中波波长在3000－2000米，且采用了调幅（AM）制式，**主要靠地波和天波传送**。由于是调幅模式，带宽较小，所以无法传输高保真节目，这就是为什么中波广播声音没有调频清晰明亮的缘故，但获取声音信息还是绰绰有余的。好的接收设备可以通过补偿电路很好地弥补这一点。中波强台接收非常稳定。大地对中波的吸收较强，所以**只能传播200~300公里**。

[全国中波AM电台收听频率表](https://www.fanly.net/article/bMzgu3ZWFk)

### 短波（SW）

短波是指频率在3MHz（兆赫兹）~30MHz的电磁波，因为能被电离层反射，而广泛用于远距离通信，因而国际广播通常都位于短波波段。短波主要是通过**天波**传送，**覆盖范围高达数百至数千公里，甚至可以达到上万公里**

在日间时频率高于12兆赫的无线电波会传播到更远的距离；而在晚间时，频率低于12兆赫则能传播到更远的距离。19米和16米波段在午后至晚间效果较好，49米和41米波段在晚间至早晨效果较好；冬季日光时期较短，电离层的电子含量偏低，不利于短波传播；太空天气，太阳耀斑爆发，太阳风，太阳黑子等频繁活动，有时会导致短波广播全面中断。

由于电离层的不稳定性，所以声音也会出现忽大忽小不够稳定的现象，所以**AGC（自动增益）电路的补偿尤为重要**。

[短波广播电台分布表](https://blog.xmgspace.me/archives/international-shortwave-to-cn.html)

[不同的短波频率和时刻对应的短波频率表](https://shortwaveschedule.com/index.php?now)

短波的频率和电台都是不固定的，打一枪换一个地方是常态，必须在精确的时间收听精确的频率才能收听到相应的内容，否则就是一片噪声，或者干扰，什么都收不到。

[国家空间天气监测预警中心](https://www.weather.com.cn/space/)

[QRZ](https://www.qrz.com/)是一个无线电爱好者的论坛，可以查询太阳活动。

### 调频（FM）

频率在87~108 MHz（兆赫）。这种广播可传输高保真的节目。在调频发射机中允许将最大频偏限制在75KHZ。我国的调频频率规定范围为87--108MHZ。学校一般在70--87MHZ之间，一般调频收音机接收不到信号，但支持校园广播的收音机就能收到。调频的覆盖范围非常有限，**只能传输几十公里**，调频主要用于地市级广播，或近距离通信。

### 航空波段（AIR）

航空波段是航空无线电频段，航空通信使用高频2-30MHz和甚高频118-135.975MHz，航空波段使用调幅制式（AM）。

一般需要到机场附近才能收到，需要具有航空波段的收音机。请时刻遵守《中华人民共和国无线电管理条例》，不得传播、公布或者利用航空通讯无线电信息。

[航空无线电频率](https://blog.path8.net/archives/7742.html)

### 单边带（SSB）

单边带本来是一种调制方式，现在主要用于特殊电台和本地电台爱好者使用。
