# 差分

差分数组记录的是“相邻元素之间的变化量”。

```
原数组   a   = [3, 1, 4, 1, 5]
差分数组 d   = [3, -2, 3, -3, 4]
                ↑  ↑   ↑   ↑   ↑
                a[0] a[1]-a[0] a[2]-a[1] a[3]-a[2] a[4]-a[3]
```

**定义：**
- `d[0] = a[0]`
- `d[i] = a[i] - a[i-1]` （`i >= 1`）

反过来，原数组就是差分数组的**前缀和**：
```
a[i] = d[0] + d[1] + ... + d[i]
```

“差分”和“前缀和”互为逆操作的关系：

- **前缀和**：把原数组变成“累积和”
- **差分**：把原数组变成“相邻差”

## O(1) 区间加问题

假设要把 `a[l..r]` 每个元素都 `+val`：

```
操作前：  d[l-1], d[l], d[l+1], ..., d[r], d[r+1], ...

只需要：
  d[l]   += val
  d[r+1] -= val    (如果 r+1 没越界)
```

## 第三步：完整代码模板

```cpp
    // 1. 读入原数组，同时构建差分数组
    vector<int> a(n);
    vector<int> d(n + 1, 0);  // 多开一位，方便 r+1 操作
    
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    // 构建差分：d[i] = a[i] - a[i-1]
    // 更简洁的做法：把 a[i] 看成"在位置 i 加了 a[i]，在位置 i+1 减了 a[i]"
    for (int i = 0; i < n; i++) {
        d[i] += a[i];
        if (i + 1 < n + 1) d[i + 1] -= a[i];
    }
    // 或者更直接：d[0] = a[0]; for (int i = 1; i < n; i++) d[i] = a[i] - a[i-1];
    
    // 2. Q 次区间加操作
    while (q--) {
        int l, r, val;
        cin >> l >> r >> val;
        l--; r--;           // 转成 0-indexed（如果输入是 1-indexed）
        d[l] += val;
        d[r + 1] -= val;
    }
    
    // 3. 一次性求前缀和，还原最终数组
    for (int i = 1; i < n; i++) {
        d[i] += d[i - 1];
    }
    
    // d[0..n-1] 就是最终结果
    for (int i = 0; i < n; i++) {
        cout << d[i] << " ";
    }
```

### 更常用的简洁写法（省去显式构建差分的步骤）

```cpp
vector<int> d(n + 2, 0);  // 多开点防越界

// 如果原数组有值，先"加"进去
for (int i = 0; i < n; i++) {
    d[i] += a[i];
    d[i + 1] -= a[i];
}

// Q 次区间加
while (q--) {
    int l, r, val;
    cin >> l >> r >> val;
    l--; r--;  // 转 0-indexed
    d[l] += val;
    d[r + 1] -= val;
}

// 还原
for (int i = 1; i < n; i++) {
    d[i] += d[i - 1];
}
// d[0..n-1] 即答案
```

## 第四步：差分 vs 前缀和——何时用谁

| 操作 | 前缀和 | 差分 |
|------|--------|------|
| 区间求和 | **O(1)** ★ | O(n)（要还原再求） |
| 区间加减 | O(n)（逐个改） | **O(1)** ★ |
| 单点查询 | O(1) | O(1)（前缀和还原后） |
| 单点修改 | O(1) | O(1) |

**一句话总结：**
- 多次查询区间和 → **用前缀和**
- 多次区间加减 → **用差分**
- 两者经常配合使用

## 二维差分（进阶，但竞赛常考）

一维差分是“区间加”，二维差分是**“子矩阵加”**。

对于子矩阵 `(x1, y1)` 到 `(x2, y2)` 整体 `+val`，在差分矩阵上做 **4 个操作**：

```cpp
// 二维差分：子矩阵 (x1,y1)-(x2,y2) 整体 +val
// 注意：用 1-indexed
d[x1][y1]     += val;
d[x1][y2 + 1] -= val;
d[x2 + 1][y1] -= val;
d[x2 + 1][y2 + 1] += val;
```

**为什么是这 4 个点？**（和一维同理）
- `d[x1][y1] += val`：从 `(x1,y1)` 开始，右下方所有点的前缀和都 `+val`
- `d[x1][y2+1] -= val`：抵消 `y > y2` 的部分
- `d[x2+1][y1] -= val`：抵消 `x > x2` 的部分
- `d[x2+1][y2+1] += val`：双重抵消的部分要加回来（**容斥原理**）

### 还原二维前缀和

```cpp
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        d[i][j] += d[i-1][j] + d[i][j-1] - d[i-1][j-1];
    }
}
```

这就是洛谷 P3397 地毯的解法。

## 第七步：差分的常见变体和技巧

### 变体 1：差分 + 贪心

很多“区间覆盖”问题可以用差分：

> 问题：有 n 个灯泡，Q 个开关，每个开关翻转 `[l, r]` 范围内的灯泡。问最终哪些灯泡是亮的？

**思路**：差分数组记录翻转次数，最后奇数次 = 亮，偶数次 = 灭。

### 变体 2：差分数组上的二分

> 问题：每次操作给一个区间 +1，问第 k 大的值是多少？

**思路**：差分还原后排序，或配合其他数据结构。

### 变体 3：坐标压缩 + 差分

当 `l, r` 的范围很大（如 `10^9`）但操作数 Q 很小（如 `10^5`）时：
- 把所有出现过的坐标排序去重（坐标压缩）
- 在压缩后的坐标上做差分
- 这样数组大小从 `10^9` 变成 `2*10^5`

## 速记卡片

```
差分数组：  d[i] = a[i] - a[i-1]
区间加：    d[l] += val; d[r+1] -= val;  // O(1)
还原：      for(i=1; i<n; i++) d[i] += d[i-1];  // 前缀和
二维区间加：四个角加减（容斥）
本质：      差分和前缀和互为逆操作

识别信号：
  - "区间加减" → 差分
  - "区间求和" → 前缀和
  - "区间加减 + 最后问结果" → 差分 + 最后还原
```
