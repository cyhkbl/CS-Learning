# 前缀和

**前缀和**（Prefix Sum）是一种预处理技术：对于原数组 `a[0], a[1], ..., a[n-1]`，定义前缀和数组 `pref`，其中  
`pref[0] = 0`  
`pref[i] = a[0] + a[1] + ... + a[i-1]`（即前 i 个元素的和）。

这样，任意子数组 `a[l] + a[l+1] + ... + a[r-1]` 的和可以 **O(1)** 计算：  
`sum(l, r) = pref[r] - pref[l]`。

## 前缀和的作用

- **快速求区间和**：将 O(n) 的求和操作降为 O(1)。
- **配合二分查找**：当需要寻找满足区间和 ≤ 某个阈值的最大区间时，可以枚举左端点，对右端点二分。
- **配合哈希表**：处理“和为 k 的子数组个数”等问题（前缀和 + 哈希表记录出现次数）。

---

## 题型

如果题目涉及 **连续子数组的求和**，并且需要多次或高效地查询不同的区间，或者需要配合二分/哈希表来寻找满足条件的区间，那么前缀和往往是必需的。

### 多次查询任意区间的和
比如：给定数组和 Q 个询问 `(l, r)`，每次回答 `a[l..r]` 的和。  
直接用前缀和 O(1) 回答，否则每次 O(n) 会超时。

### 寻找最长的连续子数组，使其和 ≤ 某个值（如本题）
典型解法：前缀和 + 二分，或双指针（滑动窗口）。

### 和为 k 的子数组个数
用哈希表记录每个前缀和出现的次数，遍历时查找 `pref[i] - k` 的出现次数。

### 二维区域和查询（二维前缀和）
如求矩阵中某个子矩阵的元素和。

### 差分数组的逆运算
差分数组可以快速进行区间加减，而前缀和可以还原原数组。
