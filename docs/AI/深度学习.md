# 深度学习

## Pytorch

张量（Tensor）是最基础的数据单元。它可以简单理解为一个支持 GPU 加速计算的多维数组。

`torch.shape` 属性：查看张量的维度信息

`torch.numel` 属性：查看张量中元素的总数

`torch.reshape()` 方法：改变张量的形状，如果你有一个 2×3 的矩阵（总共 6 个元素），你可以将其 reshape 为 (1,6)、(3,2) 或 (6,1)。

`torch.tensor(<list>)` 方法：将一个 Python 列表转换为张量。例如，`tensor([1, 2, 3])` 会创建一个包含元素 1、2 和 3 的张量。

`torch.arange(<start>, <end>)` 方法：生成一个包含从 start 到 end（不包括 end）的等差数列的一维张量。例如，`arange(0, 5)` 会生成一个包含 0、1、2、3 和 4 的张量。
