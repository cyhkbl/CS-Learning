# 反代与API统一管理

## Antigravity Manager

1. 访问[Antigravity Manager](https://github.com/lbjlaq/Antigravity-Manager)进行下载。
2. 登录Google账号后，Antigravity Manager 会把 Antigravity 的对话反代为OpenAI API格式。`Base URL`为：<http://127.0.0.1:8045>

## CC Switch

1. 首先下载[CC Switch](https://github.com/farion1231/cc-switch)，配置你的模型提供商和模型。
2. 然后把 Proxy 打开即可覆盖ANTHROPIC_AUTH_TOKEN"和"ANTHROPIC_BASE_URL"。
