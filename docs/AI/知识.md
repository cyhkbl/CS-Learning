# 知识

## 命题逻辑

- 命题符号：命题是可以被判定为真或假的陈述。用符号（如P、Q、R）表示命题。
- 逻辑连接词：命题逻辑使用逻辑连接词来构建复杂的命题。有5个主要的连接词：
  - 合取（AND，∧）：P ∧ Q 表示“P且Q”。只有当P和Q都为真时，P ∧ Q才为真。
  - 析取（OR，∨）：P ∨ Q 表示“P或Q”。当P或Q任一为真时，P ∨ Q为真。
  - 否定（NOT，¬）：¬P 表示“非P”。
  - 条件（IMPLIES，→）：P → Q 表示“如果P则Q”。当P为真时，Q被期望为真，如果不是，则P → Q为假；但如果P为假时，我们无法对Q的真假做出断言，P → Q总为真。
  - 双条件（IF AND ONLY IF，↔）：P ↔ Q 表示“P当且仅当Q”。当P和Q的真假相同时，P ↔ Q为真。

## 模型

模型会为命题符号分配真值，从而决定复合命题的真值。

## 知识库

知识库是一个包含多个命题的集合，表示已知的信息。知识库中的命题可以用来推导新的信息或验证查询的真值。            

- 蕴含：如果命题α蕴含命题β（记作α ⊨ β），那么在所有使α为真的模型中，β一定为真。
