# SMILES

Simplified Molecular Input Line Entry System（简化分子线性输入规范）是一种用**一行字符串**来描述分子结构。

* **原子（Atoms）**：用元素符号表示，如 `C` (碳), `N` (氮), `O` (氧)。
* **单键/双键/三键**：
* 单键通常省略（如 `CC` 是乙烷）。
* 双键用 `=` 表示（如 `C=C` 是乙烯）。
* 三键用 `#` 表示（如 `C#N` 是氢氰酸）。
* **分支（Branches）**：用括号 `()`表示。
* 例如，异丙醇表示为 `CC(O)C`。
* **环状结构（Rings）**：用**数字**标记环的开口和闭合。例如苯环：`C1=CC=CC=C1`（数字1代表环从这里开始并在这里结束）。
