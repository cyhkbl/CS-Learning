# 钱包

钱包在本地（手机/电脑）保存或管理的是**助记词/私钥**（通常加密存储在设备里）。由该私钥控制的**地址（公钥）**、该地址的**余额、NFT、交易记录**等。钱包 App 是**生成/管理私钥（助记词）并签名交易**的工具。

## 地址

主流钱包用的是 **HD 钱包**（分层确定性钱包）机制：
同一份助记词 → 生成同一个“主种子” → 按不同 **derivation path（派生路径）** 为不同链/不同账户派生出不同密钥对。

- EVM 常见派生路径：`m/44'/60'/0'/0/0`
- Solana 常见派生路径：常见实现有 `m/44'/501'/0'/0'` 等（不同钱包可能略有差异）

同一个助记词/种子（seed phrase）通过不同的派生路径/椭圆曲线算法，派生出 **不同的私钥**（因此也会有不同的公钥与地址）。

EVM 链（BNB Chain、Ethereum 等）使用**secp256k1**签名曲线，由公钥做 Keccak 哈希后取后 20 字节，因此 BNB Chain 和以太坊的地址通常可以**完全一样**（同一套 EVM 私钥对应同一个 0x 地址），只是“在哪条链上使用”不同。

Solana 使用**Ed25519**签名曲线，地址直接就是公钥的 Base58 编码，因此 Solana 地址和 EVM 地址**不可能一样**。
