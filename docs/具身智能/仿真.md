# 仿真

## URDF 建模

URDF（Unified Robot Description Format，统一机器人描述格式）是一种基于 XML 的文本格式，专门用来描述机器人的几何结构、运动学、动力学属性以及碰撞检测模型。

简单来说，使用 URDF 建模，就是**用代码组装一个机器人**。你需要定义好每个部件的形状、大小、颜色、质量，部件之间的连接方式（是固定还是旋转），以及关节的转动范围等信息。

一个标准的 URDF 模型通常包含两部分：

-   **链接**：机器人的刚体部件，比如手臂、轮子、底座。需要定义它的视觉形状（用于显示）、碰撞形状（用于物理计算）和惯性参数（质量、转动惯量）。
-   **关节**：连接两个链接的活动部件，比如旋转关节、平移关节、固定关节。需要定义它的类型、父子关系、转动轴、运动极限等。
