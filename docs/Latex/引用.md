# 引用

## 引用格式

**对于期刊文章：**

```latex
@article{smith2009deep,
  title={},
  author={Smith, John and Doe, Jane},
  journal={},
  volume={2019},
  pages={1--13},
  year={2019},
  publisher={Hindawi}
}
```

**对于网站/杂项：**

```latex
@misc{wiki:red_star_over_china,
  title = {红星照耀中国},
  author = {维基百科编者},
  organization = {维基百科，自由的百科全书},
  year = {2025},
  note = {[Online; accessed 2025-11-29]},
  url = {https://zh.wikipedia.org/wiki/%E7%BA%A2%E6%98%9F%E7%85%A7%E8%80%80%E4%B8%AD%E5%9B%BD}
}
```

`misc`类型适用于无法归类为其他类型的引用，如网站、报告等。

`wiki:xxx`是引用标签，可以根据需要自定义 xxx 部分，以便在正文中方便地引用。

`year`为页面上标注的“最后编辑年份”。

`note`用于注明这是一个在线资源，并记录了访问该页面的具体日期（2025-11-29）。

## 脚注

### 基本脚注

```latex
这句话需要脚注说明\footnote{这里写脚注内容。}
```

### 无编号脚注

```latex
\footnotetext{这是一条不带编号标记的脚注内容。}
```

只在页脚放脚注文字，不在正文里放标记。

### 用符号而不是数字做脚注标记（如 * † ‡）

```latex
\renewcommand{\thefootnote}{\fnsymbol{footnote}}
正文\footnote{符号脚注。}
\renewcommand{\thefootnote}{\arabic{footnote}}
```

### 在章节标题里加脚注

直接 `\section{... \footnote{...}}` 可能报 warning/出错。

```latex
\section{标题\texorpdfstring{\footnote{脚注内容}}{}}
```
