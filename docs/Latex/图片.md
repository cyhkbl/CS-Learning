# 图片

## 1. 加载宏包（有时.sty文件中已包含）
```latex
\usepackage{graphicx}
```

## 2. 图片插入语法

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.6\textwidth]{image.jpg}
    \caption{图片标题}
    \label{fig:image1}
\end{figure}
```

图片位置：按照从前往后的顺序尝试放置图片

`h` - 当前位置

`t` - 页面顶部

`b` - 页面底部

`p` - 单独一页

图片尺寸：

```latex
\includegraphics[width=0.5\textwidth]{image.jpg}     % 相对文本宽度
\includegraphics[height=3cm]{image.jpg}             % 固定高度
\includegraphics[scale=0.7]{image.jpg}              % 缩放比例
\includegraphics[width=0.6\textwidth, height=4cm, keepaspectratio]{image.jpg}    % 保持宽高比
```

使用`\listoffigures`命令可以生成图片目录。
