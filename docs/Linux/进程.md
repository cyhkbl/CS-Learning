# 进程

## 查看所有进程
   
```bash
ps aux
```

## 搜索特定进程

```bash
ps aux | grep 进程名或关键词
```

## 动态查看进程（实时监控）

```bash
top
```

实时显示进程状态，按`q`退出。可按`Shift + P`按CPU排序，`Shift + M`按内存排序。

## 更强大的实时监控工具(htop)

提供更友好的交互式界面。

```bash
htop
```

### 结束进程

```bash
kill PID号          # 正常结束
kill -9 PID号       # 强制结束
```
