# 权限

## sudo

> `sudo`（superuser do 的缩写）是 Linux 和类 Unix 系统中用于以超级用户（root）权限执行命令的工具。它允许普通用户在不切换到 root 用户的情况下，临时获得管理员权限来执行特定任务。

需要修改**系统级文件、安装/删除软件、更改系统配置、管理服务或访问受保护目录**时，通常需要 `sudo`。

1. **安装/卸载软件**（通过 `apt`、`snap` 等包管理器）
   ```bash
   sudo apt install neofetch  # 需要 sudo
   ```
2. **修改系统配置文件**（如 `/etc` 目录下的文件）
   ```bash
   sudo nano /etc/hosts
   ```
3. **管理系统服务**
   ```bash
   sudo systemctl restart nginx
   ```
4. **访问受限目录或文件**（如 `/root`、`/var/lib`）
   ```bash
   sudo ls /root
   ```
