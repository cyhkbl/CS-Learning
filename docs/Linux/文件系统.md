# 文件系统

## 查看存储容量

`df` 是最常用的查看磁盘使用情况的命令。添加 `-h` 参数可以使输出更易读（人性化格式显示，以 GB、MB 等单位显示）：

```bash
df -h
```

输出示例：

```
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1        20G   10G   9G  53% /
tmpfs           448M     0  448M   0% /dev/shm
```

使用 `du` 命令可以查看特定目录的磁盘占用情况。例如，查看当前目录：
```bash
du -sh
```
